当用户要求生成、创建、绘制、设计、编辑、改风格或以其他方式产出图片时，使用本工具。这是 API function 工具，不是 shell 命令：直接通过工具接口调用 GenerateImage，绝不要在 Shell 里运行 `GenerateImage`。在收到成功的 GenerateImage 工具结果之前，不得声称图片已生成；也绝不能用 WebSearch、截图或现成的网络图片冒充生成结果。支持从文本 prompt 生成 AI 图片（文生图），或对参考图做编辑/改风格（图生图），由平台在服务端执行。每张生成的图片按平台设定的引擎单价扣除组织积分；余额不足时调用会被直接拒绝，失败不扣费。引擎："gpt-image-2"（默认，旗舰画质）和 "seedream"（火山方舟 Ark 备选）；两者价格不同，平台绝不自动切换引擎——仅在用户同意后才显式传 model:"seedream" 重试。图生图时用工作区相对路径传 refImages（如 "uploads/photo.jpg"）。生成的 PNG 文件由服务端写入当前工作区的 assets/generated/<date>/，结果 JSON 返回其相对路径和扣除的积分；用 markdown 图片语法向用户展示每张图，如 ![描述](assets/generated/20260715/img-xxxxxxxx.png)。每张图生成约需 10-60 秒；除非用户要求多个变体，否则用 count=1。
