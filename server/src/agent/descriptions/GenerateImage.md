Generate AI images from a text prompt (text-to-image), or edit/restyle reference images (image-to-image), executed server-side by the platform. Each generated image deducts organization credits at a per-engine price set by the platform; the call is rejected up front when the balance is insufficient, and nothing is charged on failure. Engines: "gpt-image-2" (default, flagship quality) and "seedream" (Volcengine Ark alternative); they are priced differently and the platform never switches engines automatically — retry with an explicit model:"seedream" only if the user agrees. Pass refImages as workspace-relative paths (e.g. "uploads/photo.jpg") for image-to-image. Generated PNG files are written by the server to assets/generated/<date>/ inside the current workspace, and the result JSON returns their relative paths plus the credits charged; display each image to the user with markdown image syntax, e.g. ![description](assets/generated/20260715/img-xxxxxxxx.png). Generation can take 10-60 seconds per image; prefer count=1 unless the user asks for variants.
